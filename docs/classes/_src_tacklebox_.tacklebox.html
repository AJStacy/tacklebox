register

  • register(name: string, callback: Function): void
    • Defined in src/TackleBox.ts:26

    register() is a public method for hooking into a hook namespace. Pass in the name of the hook and a callback function to execute on the arguments.

    The hook may allow for replacement data to be returned. If the hook does not allow for replacement data, simply return a String of "void".

    Parameters

    • name: string

      The name of the hook on which to execute the callback.

    • callback: Function

      The callback function to execute at the hook point.

    Returns void
